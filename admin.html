How to Add/Edit Products

1. Add New Product: Click the "Add New Product" button below

2. Upload Image: Select a product image (it will be stored as base64)

3. Fill Details: Enter product name, price, category, and items

4. Save: Click save - products are stored in browser localStorage and (optionally) in your GitHub repo

5. Edit/Delete: Use the action buttons in the table

Note: To publish product changes to everyone automatically, enter your GitHub repository details and a Personal Access Token in the settings below. This will commit data/products.json and trigger the GitHub Actions deploy.

GitHub Deployment Settings

Enter your repository details and a Personal Access Token (PAT) with repo contents write permission. The token is stored in your browser localStorage only.
